SylviorusSystem
========

$Sylviorus is Antispam System we are here to create a safe environment in telegram froms scammers, threats, raiders, spammers etc

Look how to use:
    
# Get your stuff done

from sylviorus import SYL

x = SYL()

syl = x.get_info(user.id)
print(x)
print(x.reason)

- Make a Better Environment
